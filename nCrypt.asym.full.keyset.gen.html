Generate a keyset. This keyset can support signing, encryption or both.

Parameters

Name Type Optional Description

curve_enc

string

 

Curve to use for the encryption keypair. Pass null to omit this for a signing only keypair.

curve_sig

string

 

Curve to use for the signing keypair. Pass null to omit this for an encryption only keypair. Please note: The keypair must be either signing, encryption or both. Neither encryption nor signing results in an exception returned as this makes no sense.

pass

string

 

The private parts of this keyset will be encrypted using this password.
