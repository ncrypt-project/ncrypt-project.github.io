change(ks, old_pass, new_pass[, sym_alg][, sym_opts])

Change the encryption options of a keypair. Change the password (to keep it the same, simply pass the same string for old and new pass), and/or the encryption options and algorithm. If algorithm and options are omitted, existing options are used.

encrypt(ks, pass[, sym_alg][, sym_opts]) → string

Encrypt a (serialized) keyset. This function only encrypts the private key information contained in the secret, leaving the public key information intact.
A keyset can / should not be encrypted twice, so if this is encrypted already, decrypt.
