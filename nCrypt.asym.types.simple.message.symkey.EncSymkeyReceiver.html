new EncSymkeyReceiver(skey)

As a receiver, retrieve data from a received encrypted symmetric key object.
You might pass an optional decryption key to decrypt the symmetric key in the constructor, or omit it to decrypt later.
Please note: If passing an instance of this class as the first argument, a clone will be returned.

decryptSymkey  function()

Decrypt the symmetric key. Please note this function does NOT return the decrypted symmetric key. It returns a boolean telling whether it could decrypt or not.
The decryption key argument can be a string (which will be used directly as a decryption key), or a shared secret object (the shared secret will be used as a decryption key).
