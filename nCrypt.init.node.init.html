initialise(onDone)

Collect random values in Node.js to initialise nCrypt. This function will make use of Node.js internal random values collector.
When initialising is done, @onDone will be called (without parameters). In this function, start calling the actual application code - nCrypt is initialised now so it can be used.
Never start nCrypt without initialising: It will be rather broken, not work correctly, and if, highly insecure in many parts of the library.
